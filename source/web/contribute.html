e (a temporary name) is an ambitious computer science and engineering project, still in the earliest stages . As an open source and public effort, participation is enthusiastically encouraged. Contributions from any and all skill levels are necessary and greatly appreciated.  There should be enough of the site already filled in to help you get a feel for the project.  Please send any suggestions, critique, or questions to coder0xff on the gmail domain. @coder0xff on twitter.

Status

Most of the work so far has been in design of the language and toolchain. The current focus is formalization of the grammar. To help this process, the Examples page could have a lot more content. Once familiar, try writing some programs and send them my way. There is no compiler, so it's up to you to get acquainted with the conventions.
