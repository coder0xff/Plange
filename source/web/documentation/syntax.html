yle="border-style:solid;border-width:1px;padding:5px;color:rgb(128,0,0);background-color:rgb(204,204,204)"> The page and any subpages are a work in progress. Significant changes and additions are expected.

The syntax of plange is context free with extensions for operator precedence and fixity and the embedding of data into a source file. The parser for plange is the purpose built parlex project. The remainder of the syntax documentation is reference material.
