The properties of operators serve as useful bases for categorization in documentation:
  • arity
  • associativity
  • precedence
  • result type - numeric, Boolean, enumerable, arbitrary
  • operand types  - numeric, Boolean, enumerable, arbitrary

  • transitive, commutative, distributive, etc properties
From this, a categorical hierarchy is constructed for interpretation as grammar rules . The resulting hierarchy was decided subjectively. Note that select symbols appear multiple times because they may be interpreted in multiple ways; that is 52 operator symbols are interpreted in 64 ways. For example, - is both "negation" and "subtraction".
