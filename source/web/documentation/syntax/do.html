invokes the nullary void function (expression)  once  and then repeatedly while or until the parenthetical evaluates to true (when coerced to bool), unless  the optional portion is omitted, in which case the function is invoked indefinitely

Syntax 

                "do"
                
                    expression
                
                [("while"|"until")
                
                    parenthetical
                
                ]
